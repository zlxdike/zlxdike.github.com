HomeGesture修改版，修复微信语音BUG，相机上划关闭BUG

发表于 2018-07-26   |   分类于 iOS Tweak   |  

需求

原版HomeGesture很容易误触，容易导致微信语音取消有bug。很烦。所以就自己修改了下。目前看来可以很有效解决这个问题。

须知

  1. 不过只测试过iPhone7 Plus。理论上支持其他机型。自行测试后果自负。是否与其他插件冲突未知。
  2. 版本号越高改动的越多，越容易与其他插件冲突。若只是需要修复微信的BUG。请使用低版本。
  3. 原版可以去VitaTaf作者Github下载
  4. 基于VitaTaf原版1.4做的修改

安装方法

iOS11越狱，不同工具越狱出来的插件加载路径不太一样。有点混乱。

  1. 可以用7-ZIP解压提取dylib文件和plist复制到到插件加载目录里即可。路径可能是其中一种。。
  • /bootstrap/Library/SBInject/
  • /usr/lib/TweakInject
  • /Library/MobileSubstrate/DynamicLibraries
  1. iOS中使用Filza安装deb包
  2. 命令行中使用dpkg -i **.deb安装
